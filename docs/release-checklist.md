- [ ] README is complete, concise, and client-facing
- [ ] Project structure is clean and intentional
- [ ] Documentation files are up to date

## Secrets and Privacy

- [ ] No API keys or tokens are committed
- [ ] Workflow credentials are placeholders only
- [ ] No personal or client-identifiable data is exposed

## Assets and Examples

- [ ] Workflow screenshot is present and clear
- [ ] Slack notification screenshot is present

## Workflow Validation

- [ ] workflow.json imports successfully into n8n
- [ ] Form -> OpenAI -> JavaScript -> IF path works
- [ ] Google Sheets mappings write correct fields
- [ ] Slack and Gmail notifications send as expected

## GitHub Readiness

- [ ] LICENSE file is present (MIT)
- [ ] Repository description is set
- [ ] Relevant topics/tags are added
- [ ] Default branch is up to date
